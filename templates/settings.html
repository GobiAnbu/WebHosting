ℹ️ How to get WhatsApp API credentials

1. Go to Meta for Developers
2. Create a new app → Select "Business" type
3. Add "WhatsApp" product to your app
4. Go to WhatsApp → API Setup
5. Copy the Temporary Access Token and Phone Number ID

⚠️ Development Mode vs Live Mode

If your app is in Development mode, you can only send messages to phone numbers added as test recipients:

→ Meta Dashboard → Your App → WhatsApp → API Setup → scroll to "To" field → click "Manage phone number list" → add recipient numbers (up to 5)

To send to any number, switch to Live mode:
→ App Settings → Basic → toggle App Mode to "Live"
(requires Meta Business Verification)

🤖 WhatsApp Bot (Auto-Reply)
✨ How it works

Members can message your WhatsApp Business number and the bot will auto-reply with chit details.

Commands members can send:
• hi / hello — Welcome message with menu
• status — Chit number, remaining, amount
• balance — Amount to pay this month
• members — List all members
• details — Full chit information
• Their name — Personal payment history
